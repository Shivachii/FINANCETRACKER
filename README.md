## Finance Tracker

This is a personal finance tracker application built with Next.js, Prisma, Clerk authentication, and Tailwind CSS. It allows users to manage their income and expenses, categorize transactions, and gain insights into their financial habits.

## Features

1. User Authentication: Secure user authentication using Clerk.
2. Transaction Management:
   Add income and expense transactions.
   Categorize transactions.
   View transaction history.
3. Category Management:
   Create and manage income and expense categories.
4. Data Visualization: (Planned/Future)
   Charts and graphs to visualize income and expenses.
5. Responsive Design: Built with Tailwind CSS for a seamless experience on various devices.

## Technologies Used

Next.js: React framework for server-side rendering and routing.
Prisma: ORM for database access.
Clerk: User authentication and management.
Tailwind CSS: Utility-first CSS framework.
Lucide React: Icon library.
Sonner: Toast notification library.
SQLite: Database. (development)
NEON Postgres : Database (production)
